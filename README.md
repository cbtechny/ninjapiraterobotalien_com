## ğŸ’» Local Development

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- A local web server (optional, but recommended)

### Running Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/cbtechny/ninjapiraterobotalien_com.git
   cd ninjapiraterobotalien_com
   ```

2. Serve the files using a local web server. Choose one of these options:

   **Using Python:**
   ```bash
   python -m http.server 8000
   ```

   **Using Node.js (npx):**
   ```bash
   npx http-server
   ```

   **Using PHP:**
   ```bash
   php -S localhost:8000
   ```

3. Open your browser and navigate to:
   ```
   http://localhost:8000
   ```

### Why Use a Local Server?

While you can open `index.html` directly in a browser, using a local server is recommended because:
- JavaScript module imports work correctly
- Dynamic content loading functions properly
- Avoids CORS issues with fetch requests
